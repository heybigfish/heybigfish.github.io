## How to Use

### Install depend

全局安装

```bash
  npm install -g hexo-cli
```

Install to Hexo blog root directory, **Not theme directory**

```bash
  npm install
```

### serve

启动服务

```bash
  hexo s
```

清除缓存

```bash
  hexo cl
```

本地发布服务

```bash
  hexo g
```

开始写做

```bash
  hexo new '新内容'
```

/source/\_posts 目录下的 [新内容.md]文件，开始编辑。

云端发布服务

```bash
  hexo g -d
```
